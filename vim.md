# vim


## Exit file
```
:q
```

## Exit file and don't save changes
```
:q!
```

## Find and replace first occurence in line
```
:s/foo/bar/
```

## Find and replace all occurences in line
```
:s/foo/bar/g
```

## Find and replace all occurences in file
```
:%s/foo/bar/g
```

## Find and delete first occurence in line
```
:s/foo//
```

## Find and delete all occurences in line
```
:s/foo//g
```

## Find and delete all occurences in file
```
:%s/foo//g 
```

## Go to beginning of next line
```
o
```

## Go to end of file
```
G
```

## Go to normal mode
```
`Esc`
```

## Select all
```
%
```

## Turn off higlighting
```
:noh 
```

## Write change to file
```
:w
```

## Write change to file and quit
```
:wq
